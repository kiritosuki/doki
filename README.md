# **doki** ✨

**doki** 是一个使用 Go 编写的开源容器运行时项目，灵感来自 Docker，但设计得更轻量、更易读、更易于学习与实验。对于想深入理解 Linux Namespace、Cgroups、镜像格式与容器机制的人来说，`doki` 是一个非常适合作为学习与二次开发的项目。

## 🐳 **名字的由来**

- "**ki**"（き）在日语中有“容器（container）”的含义
- "**doki**" 在日语语境里常作为“心跳声、怦然心动”的拟声词

> 所以 **doki = 容器 + 心跳声** → 一个“为容器而心动”的项目 ❤️

也象征这是一个正在跳动、不断成长的开源容器项目。

# ✨ 项目目标

`doki` 希望成为：

- 一个 **可阅读、可修改、可学习** 的容器实现
- 一个拥有清晰 CLI 的容器工具
- 一个适合用来理解 Docker 工作方式的参考实现
- 一个能自由扩展的实验性平台

最终你可以用 doki 实现类似：

doki run ubuntu /bin/bash

或扩展：

- rootless 模式
- 简化网络驱动
- 轻量版镜像格式
- 自定义安全策略

# 🛠 技术栈

- **Go 1.22+**
- **Linux（Ubuntu）** 用于实际容器相关系统调用
- **Namespace / Cgroups / Chroot / OverlayFS**
- **urfave/cli v3** 作为 CLI 框架

开发在 macOS 上使用 IDE，但实际执行会在 Linux（包含虚拟机）运行。
